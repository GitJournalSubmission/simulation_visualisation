Fish movement & acoustic telemetry detection modeling

This tool simulates the movement and behaviour of acoustically tagged fish within a defined space equipped with acoustic telemetry receivers. It's designed to help researchers plan field studies and test new methodologies in a controlled environment. A detailed description can be found in the accompanying manuscript (URL).

Simulation elements

Study area Define the size and depth of the study area (in meters).
Fish behaviour Each simulated fish moves within its home range and switches between active (feeding, exploring) and resting (cruising) states, influenced by individual movement parameters, social interactions with nearby fish.
Receiver array Place acoustic receivers within the habitat to detect nearby fish, simulating a real-world telemetry array used in fish tracking research. The 80% detection range can be selected and a placing error applied. Below 10% placing error the layout of the receivers can be changed to a fine-scale array layout (VPS)
Detection simulation Each fish is equipped with an acoustic tag that randomly pings in 30-90 seconds intervals. As fish move through the study area, the simulation calculates whether each receiver would detect them based on distance and detection probability.

Two modes

Real-time visualisation (default setting) Used to setup the proposed receiver array design and to observe the simulated individuals.
Fast simulation (optional) Mode to simulate multiple days by selecting the number of days and clicking run. Depending on the size of the study area, number of fish/receiver, this might take a while. This functionality is currently limited to three days to avoid RAM issues in the browser. Once the simulation is completed (indicated by the message "Fast simulation complete!" in the UI), the simulated data can be downloaded as a .zip file.
